GOALS FOR ATTENDING WEB DEVELOPMENT CLASS
Get certified Certifications can serve as proof of a developer's dedication to a specific skill.
Learn a new programming language By learning more programming languages, a software developer can have more opportunities to work on various projects, regardless of the project's purpose or language
Improve soft skills Soft skills can be things like:
  • communication and collaboration with team members.
  • problem solving
  • teamwork ethic
Become an expert in a domain Becoming an expert may involve additional schooling, practice or certification.
learn new technologies that are relevant for web development such as HTML5/CSS3.
gain experience in working collaboratively within a group environment.
